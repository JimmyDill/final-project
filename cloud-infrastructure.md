## cloud-infrastructure

## to create an instnace in ec2
1. clicke launch instance
2. select you r operating system like Amazon Linux 2 AMI.
3. select your instance type
4. create a key pair
5. set up a security group
6. selct launch the instance.

## To create a bucket
1. navagate to your S3 dashboard
2. select create bucket
3. configure settings as required, otherwise leave as is.
4. clicke create bucket

## Allow public access using your bucket
1. click permissions tab
2. clicke bucket policy
3. click edit
4. add the required code.
